Juegos por ahora (21/12/22)

Dados primos!
Se tira un par de dados con valores primos. 2 jugadores, la primer tortuga en llegar 
a la zona delimitada con un círculo gana! 
Cada jugador tira el par de dados una vez con ENTER, ambos números se suman y se 
avanza esa cantidad. Si la verde llega primero le queda un lanzamiento a la azul
y termina el juego.
La idea es que vean los números primos y comiencen a relacionarse y recordarlos mejor.
